Linux:


Click the download, your computer will install the package right away.


This will create a new user in the system for you and install OpenXenManager.


Press "ctr-alt-t" in order to spawn a terminal


First Command: "sudo cd Downloads/" -----This allows you to enter into the Downloads Folder


Second Command: "sudo chmod +x run.sh" -----"chmod" will change permissions of the file in order to download your file


Third Command: "sudo ./run.sh" -----This is the command that will actually start the process of downloading Xen Manager


You now can start the app and log in with your username and password, using the command ./openxenmanager.
